# Area Formula Calculator

This is a C++ program that calculates the area of different shapes. I created this project while learning C++ to practice using variables, user input, functions, and mathematical formulas.

## Shapes Included

* Circle
* Rectangle
* Triangle
* Trapezoid

## How It Works

The user selects a shape from a menu and enters the required measurements. The program then calculates and displays the area.

### Formulas Used

Circle:
Area = π × radius²

Rectangle:
Area = length × width

Triangle:
Area = (base × height) / 2

Trapezoid:
Area = ((base1 + base2) × height) / 2
